# Modern & Sophisticated Website Color Palette

Based on the desired brand personality---**modern, sophisticated,
professional, easy to use, warm, and light**---this palette emphasizes
restrained color usage, warm neutrals, and a single confident accent.

## Color Distribution

-   **60%** --- Warm White (`#FCFCFB`) and White (`#FFFFFF`) backgrounds
-   **30%** --- Neutral grays for typography, cards, borders, and
    supporting UI
-   **10%** --- Indigo (`#4F46E5`) for calls-to-action and interactive
    elements

## Usage Guidelines

### Backgrounds

-   **Page:** `#FCFCFB`
-   **Cards & Forms:** `#FFFFFF`
-   **Alternate Sections:** `#F7F7F5`

### Navigation

-   Logo: `#1E293B`
-   Links: `#475569`
-   Active Link: `#4F46E5`
-   Hover Background: `#EEF2FF`

### Buttons

**Primary** - Background: `#4F46E5` - Hover: `#4338CA` - Text: `#FFFFFF`

**Secondary** - Background: `#FFFFFF` - Border: `#E7E7E3` - Text:
`#1E293B` - Hover Background: `#F7F7F5`

**Text Buttons** - Text: `#4F46E5` - Hover Background: `#EEF2FF`

### Forms

-   Input Background: `#FFFFFF`
-   Border: `#E7E7E3`
-   Focus Border: `#4F46E5`
-   Focus Ring: `#EEF2FF`

### Typography

-   Headings: `#1E293B`
-   Body: `#475569`
-   Secondary: `#64748B`
-   Placeholder: `#94A3B8`

### Cards

-   Background: `#FFFFFF`
-   Border: `#E7E7E3`
-   Hover Shadow: Soft neutral shadow
-   Selected State: `#EEF2FF`

### Status Colors

-   Success: `#16A34A`
-   Warning: `#D97706`
-   Error: `#DC2626`

## Brand Feel

This palette creates a brand that feels: - Modern - Sophisticated -
Warm - Professional - Easy to use

It achieves this by using warm neutrals, restrained color accents, and
high-contrast typography.
